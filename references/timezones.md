# IANA Timezone Reference

Complete list of all IANA timezones available in this Python install. 
Use these exact strings (case-sensitive, including underscores) when calling `set_timezone.py`.

If you don't see a region the user lives in, they should pick the nearest major city in their UTC offset. The full canonical list is also at https://en.wikipedia.org/wiki/List_of_tz_database_time_zones.
